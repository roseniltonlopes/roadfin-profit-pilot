🚗 RoadFin — Profit Pilot

Aplicativo mobile focado em ajudar motoristas de aplicativo a entenderem o lucro real das corridas, controlarem custos operacionais e tomarem decisões financeiras mais inteligentes.

📌 Sobre o projeto

O RoadFin nasceu a partir de uma dor real observada durante um período trabalhando como motorista de aplicativo.

O principal problema percebido era que muitos motoristas sabem quanto faturam, mas não conseguem calcular:

lucro real
custo operacional
gasto por quilômetro
reserva para manutenção
impacto do combustível
depreciação do veículo
metas financeiras sustentáveis

O objetivo do projeto é transformar dados financeiros do motorista em informações simples e visuais para tomada de decisão diária.

🚀 Objetivos do app

Controlar ganhos diários
Registrar despesas operacionais
Calcular lucro líquido real
Calcular custo por KM
Acompanhar combustível e manutenção
Criar metas financeiras
Gerar indicadores inteligentes
Facilitar gestão financeira para motoristas de app
